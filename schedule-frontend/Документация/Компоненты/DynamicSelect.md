# Описание
Файл: `src\components\Modal\DynamicSelect.jsx`

Компонент `DynamicSelect` — это выпадающий список, который динамически загружает опции из API. Используется в [[Модальная форма с данными]] для полей, значения которых получаются из внешних источников.

**Зависимости:**
- [[useApiData]]

# Пропсы
## Обязательные
### fieldConfig
Объект конфигурации поля, содержащий:
- `apiFunction` — функция API для получения данных
- `labelField` — поле или массив полей для отображения метки
- `valueField` — поле, используемое в качестве значения опции
- `name` — имя поля
- `placeholder` — текст-заполнитель
- `required` — признак обязательности
### value
Текущее выбранное значение списка.
### onChange
Функция-обработчик изменения значения.
## Опциональные
### disabled
Блокировка взаимодействия со списком. По умолчанию `false`.

# Переменные
## Вспомогательные переменные
- **`data`** — массив данных, полученных из API.
- **`loading`** — состояние загрузки данных.
- **`error`** — ошибка, возникшая при загрузке данных.

## Переменные из fieldConfig
- **`apiFunction`** — функция API для получения данных списка.
- **`labelField`** — поле или массив полей для формирования отображаемой метки.
- **`valueField`** — поле, используемое в качестве значения (`value`) опции.

# Функции
## getDisplayLabel(item, labelField)
Вспомогательная функция для формирования отображаемой метки из нескольких полей.

**Параметры:**
- `item` — объект данных из API
- `labelField` — поле (строка) или массив полей для отображения

**Возвращает:** Строку с объединёнными значениями полей через пробел.

**Пример:**
```javascript
// Если labelField = ['surname', 'name', 'fathername']
// Вернёт: "Иванов И.И."
```

# Хуки
## useApiData
Кастомный хук для загрузки данных из API. Вызывается с функцией `apiFunction` из конфигурации поля.

# Базовая структура
```
DynamicSelect
└── select
    ├── option (placeholder)
    └── option[] (данные из API)
```

## Элементы:
- **select** — HTML-элемент выпадающего списка
- **option (placeholder)** — первая опция с текстом "Выберите..." или "Загрузка..."
- **option[]** — список опций, сгенерированных из данных API

# Пример использования

```jsx
import DynamicSelect from "./Modal/DynamicSelect";

// В составе формы
<DynamicSelect
  fieldConfig={{
    apiFunction: getTeachers,
    labelField: ['surname', 'name', 'fathername'],
    valueField: 'id',
    name: 'teacher_id',
    placeholder: 'Выберите преподавателя',
    required: true
  }}
  value={formData.teacher_id}
  onChange={handleChange}
  disabled={loading}
/>
```

# Обёртки
Компонент используется внутри:
- [[Модальная форма с данными]] — для динамических полей типа `select`

# Состояния
## Загрузка
Показывается опция "Загрузка...", список заблокирован.

## Ошибка
Отображается сообщение: "Ошибка загрузки: {error}"

## Успешная загрузка
Показывается список опций из API с placeholder и значениями.
