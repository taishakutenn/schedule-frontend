# Описание
Файл: `src\components\Modal\Modal.jsx`
Базовый компонент для отображения модальных окон. 
# Пропсы
## Обязательные
### isOpen
`boolean` Управляет видимостью модального окна. Если `true`, модальное окно отображается, если `false` - скрывается.
### onClose
Функция обратного вызова, которая вызывается при закрытии модального окна. По умолчанию не определена. При добавлении компонента на страницу необходимо определить действия, которые будут выполнены после закрытия, например обновление данных в таблице.
### title
Заголовок модального окна. Отображается в шапке модального окна.
### children
Содержимое модального окна, которое будет отображаться в теле модального окна.
## Опциональные
### showCloseButton
`boolean` Определяет, должна ли отображаться кнопка закрытия. По умолчанию `true`.
### closeOnOverlayClick
`boolean` Разрешает закрытие модального окна при клике на оверлей (фон). По умолчанию `true`.
### closeOnEscapeClick 
`boolean` Разрешает закрытие модального окна при нажатии клавиши Esc. По умолчанию `true`.
### size
Определяет размер модального окна. Доступные варианты: `sm` (маленький), `md` (средний - по умолчанию), `lg` (большой), `xl` (очень большой), `xxl` (самый большой).
```css
.modal-sm {
  width: 300px;
  max-width: 90%;
}
.modal-md {
  width: 500px;
  max-width: 90%;
}
.modal-lg {
  width: 700px;
  max-width: 90%;
}
.modal-xl {
  width: 900px;
  max-width: 95%;
}
.modal-xxl {
  width: 1100px;
  max-width: 95%;
}
```
# Переменные
## modalRef
Реф для доступа к DOM-элементу модального окна
## overlayRef
Реф для доступа к DOM-элементу оверлея (фона)
# Функции
## handleEsc(e)
 Обработчик нажатия клавиши Escape. Закрывает модальное окно при нажатии клавиши Esc.
## handleOverlayClick(e)
 Обработчик клика по оверлею (фону). Закрывает модальное окно при клике на оверлей, если включена соответствующая опция.
# Хуки
## useEffect
- **Хук обработки клавиши Escape** - Добавляет глобальный обработчик нажатия клавиш для закрытия модального окна при нажатии Escape. Автоматически удаляет обработчик при размонтировании компонента или изменении зависимостей.
- **Хук управления прокруткой** - Блокирует прокрутку фонового содержимого при открытии модального окна и восстанавливает её при закрытии. Обеспечивает корректное поведение при размонтировании компонента.
- **Хук фокусировки** - Устанавливает фокус на модальное окно при его открытии для обеспечения доступности.
# Базовая структура
## Внешняя обертка (overlay):
- **Контейнер оверлея** - затемненный фон, который покрывает всё окно браузера
- Имеет класс `modal-overlay`
- Обрабатывает клики для закрытия модального окна
- Содержит в себе само модальное окно
## Основное модальное окно:
- **Контейнер содержимого** - центрально выровненное окно с заданным размером
- Имеет класс `modal-content` и дополнительный класс размера `modal-${size}`
- Атрибуты доступности: `role="dialog"`, `aria-modal="true"`, `tabIndex="-1"`
- Может иметь `aria-labelledby` при наличии заголовка
## Шапка модального окна (опционально):
- **Контейнер шапки** - верхняя часть модального окна с классом `modal-header`
- Отображается только если указан заголовок (`title`) или включена кнопка закрытия (`showCloseButton`)
- Содержит:
    - **Заголовок** - элемент `<h2>` с id `modal-title` (если передан `title`)
    - **Кнопка закрытия** - крестик с классом `modal-close-button` и атрибутом `aria-label="Закрыть"` (если `showCloseButton` равен `true`)
## Тело модального окна:
- **Контейнер содержимого** - основная область для пользовательского контента
- Имеет класс `modal-body`
- Содержит переданный через `children` контент
