# Конфигурация справочников

Файл: `src/utils/tableConfig.js`

Конфигурационные файлы для работы со справочниками системы. Определяют API-функции, сообщения о загрузке/ошибках, конфигурацию полей и первичных ключей.

---

## tableConfig

Объект конфигурации для загрузки данных каждого справочника.

### Структура
```javascript
{
  [handbookKey]: {
    apiFunction: Function,
    loadingMessage: string,
    errorMessage: string
  }
}
```

### Поля
- **`apiFunction`** — функция API для получения данных (например, `getTeachers`)
- **`loadingMessage`** — сообщение, отображаемое при загрузке
- **`errorMessage`** — базовое сообщение об ошибке

### Конфигурация справочников

| Ключ | API функция | Сообщение загрузки | Сообщение ошибки |
|------|-------------|-------------------|------------------|
| `teachers` | `getTeachers` | "Загрузка преподавателей..." | "Ошибка при загрузке преподавателей" |
| `teacher_category` | `getTeachersCategory` | "Загрузка категорий преподавателей..." | "Ошибка при загрузке категорий преподавателей" |
| `plans` | `getPlans` | "Загрузка учебных планов..." | "Ошибка при загрузке учебных планов" |
| `buildings` | `getBuildings` | "Загрузка зданий..." | "Ошибка при загрузке зданий" |
| `cabinets` | `getCabinets` | "Загрузка кабинетов..." | "Ошибка при загрузке кабинетов" |
| `specialities` | `getSpecialities` | "Загрузка специальностей..." | "Ошибка при загрузке специальностей" |
| `session_type` | `getSessionTypes` | "Загрузка типов занятий..." | "Ошибка при загрузке типов занятий" |
| `groups` | `getGroups` | "Загрузка групп..." | "Ошибка при загрузке групп" |
| `streams` | `getStreams` | "Загрузка потоков..." | "Ошибка при загрузке потоков" |
| `payment_forms` | `getPaymentForms` | "Загрузка форм оплаты..." | "Ошибка при загрузке форм оплаты" |

### Пример использования
```jsx
import { tableConfig } from './tableConfig';

const { data, loading, error } = useApiData(
  tableConfig[handbook]?.apiFunction || (() => []),
  [handbook, refreshTrigger],
  !!tableConfig[handbook]
);

if (loading) {
  return <div>{tableConfig[handbook].loadingMessage}</div>;
}

if (error) {
  return <div>{tableConfig[handbook].errorMessage}: {error}</div>;
}
```

## displayFieldConfig

Конфигурация полей для отображения в сообщении об удалении.

### Структура
```javascript
{
  [handbookKey]: string[]
}
```

### Конфигурация
| Справочник | Поля для отображения | Пример сообщения |
|------------|---------------------|------------------|
| `teachers` | `["surname", "name", "fathername"]` | "Иванов Иван Иванович" |
| `groups` | `["group_name"]` | "ИВТ-21" |
| `specialities` | `["speciality_code"]` | "09.02.07" |
| `cabinets` | `["building_number", "cabinet_number"]` | "3 101" |
| `buildings` | `["building_number"]` | "1" |
| `teacher_category` | `["teacher_category"]` | "Высшая категория" |
| `payment_forms` | `["payment_name"]` | "Бюджет" |
| `session_type` | `["name"]` | "Лекция" |

### Пример использования
```jsx
import { displayFieldConfig } from './tableConfig';

const fields = displayFieldConfig[handbook] || [];
const values = fields.map(field => rowData[field]);
const message = `Вы уверены, что хотите удалить ${values.join(" ")}?`;
```
